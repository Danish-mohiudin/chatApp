# 💬 Real-time Chat App  

A real-time chat application with authentication, instant messaging using Socket.io.  

## 🚀 Live Demo  
👉 [Click here to view](https://chat-app-henna-beta.vercel.app)  

## ✨ Features  
- Real-time messaging with Socket.io  
- User authentication (login/register)  
- Online/offline status indicators  
- Responsive chat UI built with Tailwind CSS  
- Private chats and user management  

## 🛠️ Tech Stack  
- **Frontend**: React, Tailwind CSS  
- **State Management**: Redux Toolkit  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Real-time Communication**: Socket.io  
- **Authentication**: JWT
